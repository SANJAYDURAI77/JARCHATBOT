## 📖 Overview

JAR Chatbot is an AI-powered conversational assistant developed to demonstrate modern chatbot development using frontend technologies, API integration, and responsive user interface design.

The project focuses on creating an intuitive conversational experience while following modular software engineering principles. It showcases practical implementation of AI-assisted interactions, reusable components, and clean application architecture.

This repository reflects hands-on experience in building intelligent web applications using modern development tools and best engineering practices.

# 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/SANJAYDURAI77/JARCHATBOT.git
```

### Navigate to Project

```bash
cd JARCHATBOT
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```
